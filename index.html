  • React JSS

    1. Go to packages/React-JSS
    2. npm install react-jss
    3. import {createUseStyles} from 'react-jss'
    4. create function

      const useStyles = createUseStyles({ //write your styles });
    5. f.e css class "list">>

      const useStyles = createUseStyles({ list: { listStyle: 'none', margin: 0, padding: 0, }, });

    6. Create some var inside Component for using hash(ending in styles) styles as a module.css--styles

      const cssClasses = useStyles();

    7. className={cssClasses.list}>
